# Static Assets Directory

This directory contains static assets used by the music video generator:

- **fonts/**: Custom fonts for lyric overlays
- **templates/**: Video templates and presets
- **images/**: Default background images
- **themes/**: Color schemes and themes

## Structure

```
static/
├── fonts/
│   └── (custom font files)
├── templates/
│   └── (video template configurations)
├── images/
│   └── (background images)
└── themes/
    └── (color scheme definitions)
```

## Usage

Assets in this directory are loaded by the visual generation engine for customization options.
